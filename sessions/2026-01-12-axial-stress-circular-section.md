**圆截面杆轴向拉应力计算**

## 📝 真题详情

### 题目来源

### 题目描述
圆截面杆ABC轴向受力如图所示。已知：
- BC杆的直径 d = 100mm
- AB杆的直径为 2d = 200mm
- 外荷载：
  - B点受到 200kN 水平向右的力
  - C点受到 100kN 水平向右的力

**求**：杆的最大拉应力

### 选项
- (A) 40MPa
- (B) 30MPa
- (C) 80MPa
- (D) 120MPa

---

## 🔍 详细解题过程

### Step 1: 截面法求各段轴力

采用**截面法**，从右往左逐段分析：

#### ① CD段分析（C点右侧）
- 截开CD段，只有C点的100kN作用
- 轴力：**N_CD = 100kN**（拉力）

#### ② BC段分析（B、C之间）
- 截开BC段，右侧受力为100kN
- 为保持平衡，左侧需提供100kN
- 轴力：**N_BC = 100kN**（拉力）

#### ③ AB段分析（A、B之间）
- 截开AB段，右侧受力 = B点200kN + C点100kN = 300kN
- 为保持平衡，左侧需提供300kN
- 轴力：**N_AB = 300kN**（拉力）✅ **最大轴力**

---

### Step 2: 计算各段截面面积

圆形截面面积公式：A = πd²/4

#### AB段（直径 2d = 200mm）
```
A_AB = π × (2d)² / 4
     = π × (200)² / 4
     = π × 40000 / 4
     = 31,416 mm²
```

#### BC段（直径 d = 100mm）
```
A_BC = π × d² / 4
     = π × (100)² / 4
     = π × 10000 / 4
     = 7,854 mm²
```

---

### Step 3: 计算各段应力

轴向拉应力公式：**σ = N / A**

#### AB段应力
```
σ_AB = N_AB / A_AB
     = 300,000 N / 31,416 mm²
     = 9.55 MPa
```

#### BC段应力
```
σ_BC = N_BC / A_BC
     = 100,000 N / 7,854 mm²
     = 12.73 MPa  ✅ 最大应力
```

---

### Step 4: 判断最大应力

| 段 | 轴力 (kN) | 截面面积 (mm²) | 应力 (MPa) |
|:---:|:---:|:---:|:---:|
| AB | 300 | 31,416 | 9.55 |
| BC | 100 | 7,854 | **12.73** ✅ |

**结论**：虽然AB段轴力最大，但因为截面面积大（直径是BC段的2倍），应力反而较小。BC段截面细，因此应力更大。

**最大拉应力**: σ_max = 12.73 MPa

---

## ⚠️ 常见错误与陷阱

### 错误思路 ❌
- 看到AB段轴力最大(300kN)，就认为应力最大
- 忽略截面面积对应力的影响

### 正确思路 ✅
- 应力 = 轴力 / 面积
- 必须综合考虑轴力和截面尺寸
- **不能简单认为轴力大就应力大**

## 📐 解题要点总结

### 核心公式
1. **轴向拉应力公式**：σ = N / A
   - N：轴力（注意单位换算）
   - A：截面面积

2. **圆形截面面积**：A = πd² / 4

### 解题步骤
1. ✅ 用截面法求各段轴力（从自由端开始）
2. ✅ 计算各段截面面积
3. ✅ 计算各段应力 σ = N/A
4. ✅ 比较所有截面，找出最大应力

### 关键注意事项
- ⚠️ 最大应力位置 ≠ 最大轴力位置
- ⚠️ 截面突变处需要特别关注
- ⚠️ 单位换算要准确（N → kN，mm² → m²）
